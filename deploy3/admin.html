How to deploy changes

1

Copy the value above

2

Go to Netlify → your site → Site configuration → Environment variables

3

Find or create INTELX_USERS and paste the value

4

Click Save — changes take effect on the next deploy. Trigger one via Deploys → Trigger deploy.
